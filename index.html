&quot

Quot is a quote reference app built in Vue.js, which utilises ajax requests to pull quotes from a ruby-on-rails API. Users can pick a topic (motivation, creativity, love, etc) and randomly cycle through related quotes. Alternatively, users can look at all the quotes in the database and filter them down through the use of the search bar.

Grumble

Grumble is a food app for the indecisive. Users enter a suburb and restaurants in that suburb are presented one by one for users to flick through. Once signed up for an account, users can add restaurants to their shortlist or their favourites list, and view restaurants in further detail to help decide where to eat.

Grumble was created in collaboration with two other developers as a learning experience in a group setting. The front end is a react app that works with a ruby-on-rails back end through the means of ajax requests.

Hotbox

HotBox is a social media platform built to support a ubiquitous consumer product, in this case; toasters. Users can view base model toasters, and custom toasters that have been added to the database. Once signed up for an account, users can rate and review base model toasters, add their own customised versions to their account, comment on the custom versions of others and upvote and/or bookmark their favourites.
